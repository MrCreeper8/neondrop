NeonDrop Downloader

Run:
1. Double-click "Start NeonDrop App.vbs" for the native PC app.
2. Paste a video link and choose Max, Lean, or MP3.
3. Optional: double-click "Create Desktop Shortcut.vbs" once to put NeonDrop on your desktop.

Downloads save to:
Downloads\NeonDrop

Video downloads:
Max and Lean are set to save/remux as MP4 when FFmpeg is available.

Required engine:
NeonDrop uses yt-dlp. If the app says the engine is missing, double-click "Install Engine.bat" or open Command Prompt and run:
python -m pip install -U yt-dlp

Optional:
FFmpeg is needed for highest-quality merging and MP3 conversion.

Fallback:
If Windows blocks the native launcher, use "Start NeonDrop.vbs" for the browser version or "Start NeonDrop.bat" as a visible fallback.

Please only download videos you own, have permission to save, or are otherwise allowed to download.
